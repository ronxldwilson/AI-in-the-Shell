   ```bash

   ```

2. Install dependencies:

   ```bash
   pip install flask requests
   ```

3. Install Ollama and run a model:

   ```bash
   curl -fsSL https://ollama.com/install.sh | sh
   ollama run llama3
   ```

4. Run the Flask server:

   ```bash
   python3 ai_shell.py
   ```
